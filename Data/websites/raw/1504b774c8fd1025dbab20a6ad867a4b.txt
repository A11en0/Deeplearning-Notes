<h1><a id="user-content-guider" class="anchor" aria-hidden="true" href="#guider"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Guider</h1>
<p>Guider is an integrated performance analyzer.<br>
It provides most of the features needed for system performance measurement and analysis.</p>
<p>The features of Guider are as follows.</p>
<ul>
<li>Monitoring</li>
<li>Profiling</li>
<li>Visualization</li>
<li>Control</li>
<li>Logging</li>
<li>Test</li>
</ul>
<p>Guider pursues characteristics as below.</p>
<ol>
<li>Easy to use: just run without any setting and package installation</li>
<li>Measure correctly: count, time in from us, size in from byte</li>
<li>Provide all features: enough functions for experiment and analysis</li>
<li>Submit the report in detail: show as much information as possible</li>
</ol>
<p>It usually supports all platforms based on the Linux kernel as shown below.</p>
<ul>
<li>Android</li>
<li>distro (Ubuntu, CentOS, RHEL, Linux Mint, Arch Linux, ...)</li>
<li>webOS</li>
<li>ccOS</li>
<li>Tizen</li>
<li>Windows (Limited)</li>
<li>macOS (Limited)</li>
</ul>

<pre><code>Enter the following command to see all commands supported by the guider:
    $ ./guider.py --help
    $ python -m guider --help
    $ guider --help

Enter the following command to start tracing for all threads:
    # ./guider.py record -a

Enter the following command to start monitoring for all processes:
    $ ./guider.py top -a

Enter the command in the format shown bellow to see options and examples for each command:
    $ ./guider.py record -h
    $ ./guider.py top -h

Visit the following link to see the output of guider:
    - https://github.com/iipeace/guider/wiki
